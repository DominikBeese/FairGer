# FairGer — arXiv v1 Archive

This folder preserves the repository structure and materials corresponding to the [arXiv v1 version](https://arxiv.org/abs/2210.04359v1) of the paper.

For the current repository version associated with the EMNLP 2024 paper, see the [repository root](../../README.md).

## Archived contents

The archived repository contains the following elements:

- 📂 [`Data`](./Data)
  - 📂 [`Datasets`](./Data/Datasets): sentences containing a woman or migrant keyword
  - 📂 [`Human Annotated Data`](./Data/Human%20Annotated%20Data): annotations of solidarity towards women and migrants
  - 📂 [`Model Predicted Data`](./Data/Model%20Predicted%20Data): predictions of the ensemble model for all datasets
- 📂 [`Code`](./Code): code to train and apply models
- 📂 [`Analysis`](./Analysis): code to generate the plots
